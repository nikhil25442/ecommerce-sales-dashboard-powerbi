# 📊 E-Commerce Sales Dashboard - Power BI

An interactive Power BI dashboard designed to analyze e-commerce sales performance, track key KPIs, and generate actionable business insights from large-scale data.

## 📌 Project Overview
This project presents an interactive E-Commerce Sales Dashboard built using Power BI to analyze large-scale business data and generate actionable insights.

---

## 🎯 Objectives
- Analyze revenue and order trends  
- Track key business KPIs  
- Identify top-performing products and regions  
- Monitor quarterly performance  

---

## 📊 Dashboard Features
- 💰 Revenue: ₹1.25B+ analyzed  
- 🛒 Total Orders: 15K+  
- 📉 Cancellation Rate & Lost Revenue tracking  
- 📦 Product-wise revenue analysis  
- 🌍 State-wise performance insights  
- 📅 Quarterly trend visualization  

---

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- Excel  

---

## 📈 Key Insights
- Top products include MacBook Air and OnePlus  
- Maharashtra and Gujarat generate highest revenue  
- Revenue drop observed in Q4  
- Product category performance varies significantly
